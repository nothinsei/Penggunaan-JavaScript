5 Contoh Penggunaan Dasar JavaScript

1. Alert Box (Popup Pesan)

Penjelasan:
Alert box digunakan untuk menampilkan pesan popup kepada user. Menggunakan fungsi alert() dan event onclick yang akan dijalankan ketika tombol diklik.

2. Mengubah Konten HTML

Ini adalah teks awal

Penjelasan:
JavaScript dapat mengubah konten elemen HTML menggunakan document.getElementById() untuk mengakses elemen dan properti innerHTML untuk mengubah isinya.

3. Kalkulator Sederhana

Penjelasan:
Contoh ini menunjukkan bagaimana JavaScript dapat mengambil nilai dari input form menggunakan value, melakukan perhitungan, dan menampilkan hasilnya. Menggunakan parseInt() untuk mengkonversi string menjadi angka.

4. Jam Digital Real-time

Penjelasan:
JavaScript dapat menampilkan waktu real-time menggunakan objek Date() dan fungsi setInterval() yang mengupdate tampilan setiap detik (1000 milidetik).

5. Ubah Warna Background dengan Mouse

Penjelasan:
Menggunakan event onmouseover (saat mouse masuk ke area tombol) dan onmouseout (saat mouse keluar dari area tombol) untuk mengubah warna background halaman secara dinamis.
